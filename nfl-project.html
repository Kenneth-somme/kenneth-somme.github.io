NFL Offensive Evolution

Run-First to Pass-First (2002–2025)

SQL PostgreSQL Power BI

Project Overview

This project explores how NFL offensive strategy has transformed over more than two decades. Using team-level game data from 2002 to 2025, I analyzed the league-wide shift from run-heavy offenses toward pass-dominated playcalling — and investigated whether that trend is beginning to reverse in recent seasons.

The project involved building a PostgreSQL database from raw CSV data, writing complex SQL queries to extract meaningful trends, and presenting the findings through an interactive Power BI dashboard.

Objective

To identify and quantify the evolution of NFL offensive strategy from 2002 to 2025 by analyzing passing and rushing statistics over time — examining league-wide trends, team-level patterns, and the relationship between offensive approach and winning.

Features

Trend Analysis: Season-by-season tracking of passing and rushing yards per game across the entire league, revealing the long-term shift toward pass-heavy offenses.

Pass/Rush Ratio: A calculated metric showing how the balance between passing and rushing has evolved each season, making the strategic shift easy to quantify.

Team-Level Breakdown: Comparison of average passing and rushing stats per team across the full 2002–2025 period, highlighting which franchises leaned into each strategy.

Wins & Passing: Exploration of the relationship between passing volume and team success, testing whether throwing more actually correlates with winning.

Interactive Dashboard: All findings visualized in Power BI with filters, trend lines, and drill-down capabilities for deeper exploration.

                ), aggre AS (
                	SELECT
                		season,
                		team,
                		COUNT(*) AS games,
                		SUM(w_l) AS w_l,
                		SUM(pen_num) AS pen_num,
                		SUM(pen_yards) AS pen_yards,
                		SUM(interceptions_for) AS ints_for,
                		SUM(interceptions_against) AS ints_against,
                		SUM(pass_yards) AS pass_yards,
                		SUM(pass_att) AS pass_att,
                		SUM(pass_comp) AS pass_comp,
                		SUM(rush_yards) AS rush_yards,
                		SUM(rush_att) AS rush_att,
                		SUM(sack_yards) AS sack_yards,
                		SUM(sacks_forced) AS sacks_forced,
                		SUM(sacks_recieved) AS sacks_recieved,
                		SUM(points_for) AS points_for,
                		SUM(points_against) AS points_against
                	FROM sorting
                	GROUP BY season, team
                ), cleaning AS(
                	SELECT 
                		season,
                		team,
                		w_l AS wins,
                		ROUND((w_l) / games, 3) AS w_l_pct,
                		pen_num,
                		pen_yards,
                		ints_for - ints_against AS interception_differential,
                		ROUND((pass_yards::NUMERIC + sack_yards) / pass_att, 1) AS pass_yards_per_attempt,
                		ROUND(((pass_comp::NUMERIC / pass_att) * 100), 1)  AS pass_completion_percent,
                		ROUND(rush_yards::NUMERIC / rush_att, 1) AS rush_yards_per_attempt,
                		sacks_forced - sacks_recieved AS sacks_differential,
                		points_for,
                		points_against
                	FROM aggre
                )
                SELECT *,
                	CASE
                		WHEN w_l_pct > 0.500 THEN 'Winning'
                		ELSE 'Losing'
                	END AS team_record
                FROM cleaning
                ORDER BY w_l_pct DESC;

Results

The data confirmed a clear and sustained shift toward pass-heavy offenses from 2002 through approximately 2018, with passing yards per game rising significantly while rushing volume declined. However, an unexpected finding emerged in more recent seasons: the pass-to-rush ratio has begun to level off and slightly reverse, suggesting the NFL may be entering a new cyclical phase as defenses adapt to pass-heavy schemes.

At the team level, significant variation exists — some franchises have consistently leaned on the pass while others maintained run-first identities across the full period. The wins analysis showed a moderate positive correlation between passing volume and winning, though the relationship is not absolute, indicating that scheme fit and execution matter as much as raw volume.

Recommendation: Teams looking to gain a competitive edge may find value in reintroducing balanced or run-first elements as the league overcorrects toward pass defense — a classic cyclical adaptation pattern in professional football.

Project Structure

Data — Raw CSV file containing NFL team game statistics from 2002 to 2025, sourced from Kaggle.
SQL / PostgreSQL — Database built in PostgreSQL 17. Queries written using CTEs, window functions, aggregations, and subqueries to extract and transform data for analysis.
Power BI — Interactive dashboard built in Power BI Desktop, featuring trend lines, scatter plots, bar charts, and season/team filters for exploratory analysis.
